Requirements

GoalballScoreSheet is written in Java so it can be run on any platform. In order to run it Java Runtime Environment must be installed.

Downloading and updating GoalballScoreSheet

GoalballScoreSheet can be downloaded from the following link
Goalball ScoreSheet

After you click on the link choose an option to save the file to your computer. After you do this you will be able to run the GoalballScoreSheet whenever you want, if you are connected to the internet or not. However if you are connected to the internet and you decide to run the GoalballScoreSheet it will first check online to see if a newer version is available. If there is a newer version available it will automatically be downloaded.

Running the Goalball ScoreSheet

After downloading the Goalball ScoreSheet from the above link, simply double click on the GoalballScoreSheet.jnlp file to start the Goalball ScoreSheet. It will first display a warning asking you if you trust the application's digital signature. Once you click on 'Run' Goalball ScoreSheet will start. For information about using the Goalball ScoreSheet you can click here or use the application's help.
